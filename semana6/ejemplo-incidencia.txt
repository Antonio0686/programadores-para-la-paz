Ejemplo de JSON para reportar una incidencia comunitaria

Ruta:
POST http://localhost:3000/incidencia

JSON enviado:
{
  "tipo": "Iluminación pública",
  "descripcion": "La comunidad reporta que una lámpara del parque no funciona desde hace varios días."
}

Respuesta esperada:
{
  "mensaje": "Incidencia registrada",
  "tipo": "Iluminación pública",
  "descripcion": "La comunidad reporta que una lámpara del parque no funciona desde hace varios días."
}

Este ejemplo permite simular cómo un ciudadano puede enviar un reporte de manera organizada.
