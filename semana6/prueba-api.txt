Prueba de la ruta POST /registro

Para probar la ruta se envía una petición POST a:
http://localhost:3000/registro

JSON enviado:
{
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}

Respuesta esperada del servidor:
{
  "estado": "Datos recibidos",
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}

Esta respuesta muestra que el servidor recibió correctamente los datos enviados en formato JSON.
También demuestra que Express puede leer la información usando req.body.
